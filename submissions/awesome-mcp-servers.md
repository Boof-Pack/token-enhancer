## PR description

Adds token-enhancer to the Search & Web section.

token-enhancer fetches any URL and returns clean text with HTML, scripts, ads, and nav stripped. It caches results so repeat fetches are instant. Yahoo Finance drops from 704,760 tokens to 2,625. GitHub repo pages drop from 171,234 to 6,976.

No API key. No LLM. No GPU. Just Python. Works as an MCP server (Claude Desktop, Cursor) or a standalone HTTP proxy.
